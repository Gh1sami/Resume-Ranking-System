# Resume-Ranking-System

This project builds an automated and flexible CV filtering and scoring system. It uses NLP techniques (TF-IDF and Cosine Similarity) to match resumes against any job description and rank candidates by relevance score.

The system is fully customizable — simply provide your own job description and the number of top candidates you want, and it will instantly filter and rank the most relevant resumes. Built with Python on Kaggle, it can be adapted to any industry, role, or hiring context.
